            <!doctype html>

            <html>

              <head>

                <title>로그인 폼 예제</title>

              </head>

              <body>

                <form name="loginform"
                action="로그인 사이트을 입력하세요."
             method="post">

                아이디: <input name="username" 
                type="text">

                <br>

               비밀번호: <input name="password" 
                type="password">

              <input type="submit" value="로그인">

                </form>

              </body>

            </html>
            ⊙코드를 복사하여 사용하세요.

로그인 폼 코드 예제

이 코드는 웹 페이지에 로그인 폼을 표시하고, 사용자가 아이디와 비밀번호를 입력한 후 "로그인" 버튼을 클릭하면 해당 정보를 지정된 서버로 제출하는 동작을 수행합니다.

post 방식으로 제출된 데이터는 URL상에서 보이지 않기 때문에 보안적입니다.

사용자가 아이디와 비밀번호를 입력한 후 "로그인" 버튼을 클릭하면, 입력된 데이터가 서버로 전송되어 로그인 처리가 이루어집니다.
