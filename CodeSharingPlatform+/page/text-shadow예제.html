            <style>

            h1 {

              font-size: 100px;

              font-family: "Arial Rounded MT";

             }

            .shadow1 {

              color: blue;

              text-shadow: 1px 1px 10px skyblue;

             }

            .shadow2 {

              color: black;

              text-shadow: 5px 5px 3px red;

             }

            .shadow3 {

              color: white;

              text-shadow: 7px -7px 5px black;

             }

            </style>

            


            <body>

            <h1 class="shadow1">내용을 입력하세요.</h1>

            <h1 class="shadow2">내용을 입력하세요.</h1>

            <h1 class="shadow3">내용을 입력하세요.</h1>

            </body>

            ⊙코드를 복사하여 사용하세요.

CSS text-shadow 예제 코드

이 코드는 CSS를 사용하여 텍스트에 그림자 효과를 부여합니다.

shadow1 클래스는 텍스트의 색상을 파란색으로 지정하고, 텍스트 주위에 하늘색 그림자를 만듭니다.

shadow2 클래스는 텍스트의 색상을 검은색으로 설정하고, 텍스트 주위에 작은 빨간색 그림자를 만듭니다.

shadow3 클래스는 텍스트의 색상을 흰색으로 설정하고, 텍스트 주위에 검은색 그림자를 만듭니다.
