웹 폼 코드 예제

이 HTML 코드는 간단한 검색 폼을 정의하고 있습니다.

이 폼은 사용자로부터 입력을 받아 서버로 전송하는 역할을 합니다.

사용자가 텍스트 입력 필드에 값을 입력하고 "검색" 버튼을 클릭하면, 폼 데이터가 action 속성에 지정된 URL로 전송됩니다.
